# Tip Analysis and Prediction

## 📌 Project Overview
This project analyzes restaurant data and predicts customer tips using Machine Learning.

## 🛠️ Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib
- Scikit-learn

## 📊 Features Used
- Total Bill
- Size (Number of people)

## 📈 Key Insights
- Tip increases as total bill increases
- Higher tips observed on weekends (Sunday)
- Most tips fall between 2–4 range

## 🤖 Model
- Linear Regression model used for prediction

## 🎯 Result
Predicted tip for:
- Bill = 50
- Size = 3  
👉 Tip ≈ 5.88

## 📎 Conclusion
This project demonstrates how data analysis and machine learning can be used to understand customer behavior and make predictions.
